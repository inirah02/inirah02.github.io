## Overview

Developed a Flutter app aimed at reducing the onset of dementia among senior citizens using personalized cognitive algorithms.

## Impact

- Positively impacted over **350** users across the state
- Provided support for memory reinforcement through personalized activities
- Won the **Cisco ThingQbator Cohort 5** national competition
- Received a **5 Lakh seed grant** for the project

## Key Features

- Personalized cognitive exercises
- Memory reinforcement activities
- Progress tracking and analytics
- User-friendly interface for senior citizens
- Firebase backend for real-time data management
